# Swing Playground
### Autoři:
Jakub Kočí, Filip Patrman, Vít Provazník

## O projektu
Platforma s skill-based minihrami, Virtualní měna.

Minihry: Mines and Diamonds, Time split, Memory pattern crack, Precision slider

## Rozdělení práce
Aspdotnet backend: Jakub Kočí

Design: Filip Patrman

React frontend: Vít Provazník
